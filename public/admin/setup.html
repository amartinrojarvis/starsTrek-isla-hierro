⚙️ Configuración del Panel de Administración

Para habilitar la edición visual, necesitas configurar GitHub OAuth

📋 Resumen

El panel de administración usa Decap CMS (antes Netlify CMS), una herramienta que permite editar archivos Markdown directamente desde el navegador y guardar los cambios automáticamente en GitHub.

⚠️ Requisito: Necesitas configurar una OAuth App en GitHub para permitir que el CMS acceda al repositorio.

🔧 Pasos de configuración

1

Crear OAuth App en GitHub

Ve a GitHub Developer Settings y crea una nueva OAuth App:

  • Application name: StarsTrek CMS
  • Homepage URL: https://starstrek-web.vercel.app
  • Authorization callback URL: https://starstrek-web.vercel.app/admin/
2

Obtener Client ID y Secret

Después de crear la app, copia el Client ID y genera un Client Secret. Los necesitarás para el siguiente paso.

3

Elegir backend de autenticación

Tienes dos opciones:

  • Opción A (Recomendada): Usar Netlify con Git Gateway (más simple, requiere mover el hosting a Netlify)
  • Opción B: Configurar un backend OAuth propio (requiere más trabajo técnico)

✅ Alternativa: Usar Netlify (Más fácil)

La forma más simple de usar Decap CMS es alojar el sitio en Netlify en lugar de Vercel. Netlify tiene integración nativa con Decap CMS.

  1. Crea cuenta en Netlify
  2. Importa el repositorio de GitHub
  3. Configura Identity en el panel de Netlify
  4. Habilita Git Gateway
  5. ¡Listo! El CMS funcionará automáticamente
Ver documentación de Netlify
